Настройка JETCALC

Для настройки системы выполните следующие шаги:

  1. Создайте организацию.
  2. Создайте объект учета.
  3. Создайте пользовательские роли и типовые пропуска.
  4. Создайте пользователей, вышлите приглашения на вход в систему.
  5. Настройка периодов сбора данных
  6. Создайте документы
